# 功能规划（按模块）

## A. 启动与入口
- 启动页/加载：资源预加载、进度提示、进入按钮
- 主页导航：开始涂色、进入海洋、我的作品、设置/帮助

## B. 模板库
- 模板列表：分类/难度/搜索（可选）
- 模板详情（轻量）：预览、区域数、开始按钮

## C. 涂色编辑器（核心）
- 画布：SVG 分区渲染（或转换为可点击区域）
- 上色：点击区域填色（支持连续快速填色）
- 工具条：撤销/重做、橡皮、清空、缩放、返回
- 调色板：固定色 + 最近使用（可选）
- 自动保存：定时与关键操作后写入本地
- 作品导出：生成纹理图（PNG）+ 元数据（区域颜色映射）

## D. 完成与入海
- 完成预览：确认/继续编辑
- 入海流程：创建“海洋角色实体”（纹理、碰撞盒、速度、路径）

## E. 海洋世界（展示与互动）
- 场景渲染：背景层/中景层/前景层、粒子（泡泡）
- 系统小鱼：群游算法（简化 Boids）
- 作品伙伴：沿预设路径游动、避让边界、点击反馈
- 交互反馈：点击弹层卡片（角色缩略、名称、返回编辑）

## F. 作品管理
- 列表：草稿/已入海
- 操作：继续编辑、再次入海（覆盖/新增）、删除（家长门禁）

## G. 设置与帮助
- 声音：背景音乐/音效开关
- 画面：低性能模式（关闭粒子/降低鱼数量）
- 数据：重置海洋世界、清空本地作品（家长门禁）
- 说明：操作指引、隐私说明
