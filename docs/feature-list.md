# 功能清单（可用于拆票）

## 核心（MVP）

- 应用启动与资源预加载
- 主页入口：开始涂色 / 进入海洋 / 我的作品 / 设置
- 模板列表：分类、卡片、开始涂色
- 涂色编辑器：分区点击填色、调色板、撤销/重做、缩放拖拽、清空
- 草稿自动保存与恢复
- 完成预览：确认/继续编辑
- 入海创建：生成纹理、写入海洋世界数据
- 海洋世界：背景与系统小鱼、作品伙伴游动、点击反馈卡片
- 我的作品：列表、继续编辑、删除（家长门禁）
- 设置：音效开关、低性能模式、隐私说明、清空数据（家长门禁）

## 增强（后续）

- 模板搜索/筛选更丰富（难度/主题）
- 更多互动：喂食、点击小鱼聚拢、气泡拍爆
- 奖励机制：完成贴纸/徽章（本地）
- 多场景海洋（白天/夜晚）
- 多语言与无障碍优化
