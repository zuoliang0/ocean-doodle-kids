# PRD｜海洋伙伴（Pad 横屏 Web/H5）

## 1. 背景与目标
- 为儿童提供低门槛、强反馈的“涂色 + 作品入场展示”的沉浸式体验。
- 将静态涂色结果转化为动态海洋世界中的可互动角色，提升成就感与复玩。

## 2. 目标用户
- 核心用户：3–9 岁儿童（触控操作为主）。
- 次级用户：家长/老师（引导使用、管理作品、查看说明）。

## 3. 核心体验与用户旅程
### 3.1 核心路径（MVP）
1) 进入应用（无需登录）
2) 选择海洋动物模板（海豚/海龟/章鱼/鲸鱼等）
3) 分区涂色（点击区域填色，支持撤销/重做）
4) 完成确认（预览）
5) “送入海洋”动画
6) 在海洋世界看到作品与系统小鱼一起游动，可点击触发反馈

### 3.2 次级路径
- 我的作品：本地作品集合、重进编辑、删除（需家长门禁）。
- 设置与帮助：音效开关、重置海洋世界、隐私说明。

## 4. 使用场景
- 家中：孩子自主涂色并观看海洋世界。
- 课堂/活动：引导孩子完成指定动物后展示。

## 5. 需求范围（MVP）
### 5.1 模板选择
- 模板分组（推荐/简单/进阶）。
- 模板卡片展示：缩略图、难度（区域数量）、预计时长。

### 5.2 分区涂色
- 点击分区填色（主交互）。
- 调色板：固定 12–24 色（大色块，便于触控）。
- 工具：撤销/重做、橡皮（恢复默认色）、一键清空、放大缩小、拖动画布。
- 引导：首次进入提示“点区域上色”。

### 5.3 完成与送入海洋
- 完成按钮：进入预览确认。
- 预览：展示作品，允许返回继续涂色。
- 送入海洋：过场动画（泡泡/光晕），将作品生成“海洋角色”。

### 5.4 海洋世界（动态展示）
- 场景：海底背景、漂浮气泡、海草摆动、系统小鱼群游。
- 作品角色：以涂色纹理贴图呈现，沿路径游动，避免遮挡 UI。
- 互动：点击角色 → 弹出名字/表情反馈（可用默认“我的海洋伙伴”），轻量音效。

### 5.5 本地作品管理
- 自动保存：涂色过程定期保存草稿。
- 作品列表：缩略图、创建时间、状态（草稿/已入海）。
- 删除/重置：需要家长门禁（算术题）。

## 6. 非功能性需求
- 性能：Pad 上稳定 60fps（海洋世界），涂色操作 < 50ms 响应。
- 触控：控件最小触控尺寸 ≥ 44px，避免误触。
- 离线：基础可离线运行（可选 PWA）。
- 安全与隐私：不采集个人信息；本地存储；提供隐私说明页。
- 可维护性：模板与场景资源可配置化。

## 7. 数据与指标（本地埋点可选）
- 完成率：开始涂色 → 完成确认。
- 入海率：完成确认 → 送入海洋。
- 复玩：进入海洋世界次数、作品数量。

## 8. 不做（MVP Out of Scope）
- 账号体系/云端同步。
- 社交分享/外部发布。
- 复杂自由绘画（仅分区填色为主）。
- 多人协作。

## 9. 里程碑建议
- M1：模板选择 + 分区填色 + 本地保存 + 完成预览
- M2：海洋世界展示 + 送入海洋动效 + 基础互动
- M3：作品管理完善 + 引导与家长门禁 + 离线优化
