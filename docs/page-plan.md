# 页面规划（信息架构）

## 导航结构
- `/` 启动/主页（应用入口与主分流）
- `/templates` 模板选择
- `/color/:templateId` 涂色编辑器
- `/preview/:artworkId` 完成预览与入海
- `/ocean` 海洋世界（动态展示）
- `/gallery` 我的作品（本地）
- `/settings` 设置与帮助（含隐私说明入口）

## 页面详述（样式 / 功能 / 交互）

### `/` 启动/主页
- 样式
  - 视觉基调：海洋主题、低文字负担；主色清爽（浅蓝/青绿），高对比 CTA。
  - 布局：上部品牌区（Logo/应用名可选）+ 中部主入口按钮 + 底部辅助入口。
  - 触控：所有可点击区域最小触达尺寸一致（面向儿童大手指）。
- 功能
  - 主分流：
    - 「开始涂色」→ `/templates`
    - 「我的作品」→ `/gallery`
    - 「海洋世界」→ `/ocean`
- 交互
  - 进入时：若首次启动，展示「新手引导浮层」（可跳过）。
  - 点击「开始涂色」：进入模板选择页。
  - 返回策略：作为入口页，系统返回键/手势触发二次确认退出（避免误退）。

### `/templates` 模板选择
- 样式
  - 卡片栅格：模板缩略图卡片（圆角、阴影轻、留白足）。
  - 筛选区：顶部可折叠（减少占屏）；默认不打扰。
- 功能
  - 模板列表：按主题/难度分组（例如：简单/进阶；鱼类/贝壳/潜艇等）。
  - 模板选择：点击卡片直接进入编辑，不提供大图预览。
- 交互
  - 选择模板：
    - 点击卡片 → `/color/:templateId`
    - 若是首次进入编辑器，可弹出「轻提示」说明基础操作（点按涂色/撤销）。
  - 滚动加载：骨架屏占位；失败可重试。

### `/color/:templateId` 涂色编辑器
- 样式
  - 画布居中：画布占主要区域；背景简洁，避免干扰线稿。
  - 底部工具区：大图标 + 文案（可选），“一眼懂”。
  - 底部调色盘：位于界面底部，与画笔选择合并；展示高饱和儿童友好常用色，避免过密。
- 功能
  - 基础涂色：点按封闭区域填充；支持画笔模式，画笔选择与颜色选择统一放在底部调色盘。
  - 工具：
    - 撤销/重做
    - 橡皮擦（仅画笔模式启用时出现，避免复杂）
    - 清空（高风险操作，需要二次确认）
    - 保存（自动保存 + 手动保存）
  - 作品信息：模板名、完成度提示（可选）。
- 交互
  - 进入：加载线稿；显示简洁加载态。
  - 涂色：
    - 点击区域立即反馈（轻微缩放/高亮边界/音效可选）。
    - 非封闭区域：给出轻提示“这里不能涂色哦”，不弹阻断。
  - 自动保存：涂色操作后短延迟批量写入，避免频繁存储。
  - 离开编辑器：若有未完成保存任务，显示“正在保存…”并阻止误退；保存完成再退出。
  - 完成：点击「完成」→ 生成 artworkId → `/preview/:artworkId`。

### `/preview/:artworkId` 完成预览与入海
- 样式
  - 大图展示：作品预览占屏为主；背景可用柔和海浪纹理。
  - 主 CTA：底部「入海」为主按钮；次要操作并列或收纳。
- 功能
  - 预览：展示最终作品（含线稿/不含线稿按设定）。
  - 入海：将作品加入海洋世界展示队列/本地状态（离线可用）。
  - 分享/导出（可选）：保存到相册/生成图片（若未确认需求可暂不实现，仅预留文案入口）。
- 交互
  - 点击「入海」：
    - 首次入海可展示短动画（例如作品“下沉入海”）。
    - 成功后提示并提供「去海洋世界看看」→ `/ocean`。
  - 点击「继续涂」：返回编辑器 `/color/:templateId`（保持当前作品继续编辑）。

### `/ocean` 海洋世界（动态展示）
- 样式
  - 场景：分层视差（前景气泡/中景鱼群/背景海水渐变），动画节制不眩晕。
  - 作品呈现：已入海作品转化为海洋动物在场景中游动，不使用展示相框。
  - 动态表现：海洋动物通过路径游动、轻微摆尾/摆鳍、缩放透视和前后层级变化形成接近 3D 的空间感。
- 功能
  - 展示：从已入海作品池中随机/按时间播放；支持轻量交互查看详情。
  - 海洋动物素材：加载无背景、仅包含动物本体的纹理，避免把背景一起带入海洋场景。
  - 动作系统：为每个海洋动物分配游动路径、速度、朝向、深度层级和循环动作，让动物持续“活起来”。
  - 详情入口：点击某个海洋动物 → 打开信息卡（作者=本机、时间、模板名可选）并提供「去预览」→ `/preview/:artworkId`。
  - 声音：可开关背景音与音效（与全局音量一致）。
- 交互
  - 点击海洋动物：动物轻弹起/高亮并出现详情卡；点击空白处关闭。
  - 性能保护：低端设备自动降低同时游动数量、粒子数量或帧率策略（仅描述，不做过度实现）。

### `/gallery` 我的作品（本地）
- 样式
  - 网格列表：作品缩略图卡片；支持按时间倒序。
  - 空状态：插画 + 明确按钮「去涂色」。
- 功能
  - 查看：点击作品 → `/preview/:artworkId`。
  - 管理：
    - 删除（高风险，需要二次确认）
    - 批量选择（可选；若当前需求不明确，先不实现，保持单个删除）
- 交互
  - 长按作品：进入“管理模式”（出现删除按钮/选择态）（可选，若不做则使用详情页内删除）。
  - 删除：成功后就地移除并提示“已删除”；失败给出可操作原因（存储不可用/空间不足）。

### `/settings` 设置与帮助（含隐私说明入口）
- 样式
  - 列表分组：设置项按“声音/数据/帮助/关于”分组；每项一行，点击反馈明显。
- 功能
  - 声音：背景音乐开关、音效开关、音量（若支持）。
  - 数据：
    - 存储占用显示（可选）
    - 清理缓存/重置应用（高风险，需二次确认 + 风险说明）
  - 帮助：基础操作说明（涂色、撤销、入海、我的作品）。
  - 隐私说明入口：跳转到隐私说明页或弹层（取决于现有实现方式）。
- 交互
  - 高风险操作（重置/清理）：先解释影响（会删除本地作品）→ 二次确认。
  - 无障碍/可达性：设置项点击区足够大，文案简洁。

## 全局 UI 组件
- 顶部栏（可选）：应用名、返回、音量
  - 样式：半透明/轻背景，避免压住画面重点。
  - 交互：返回遵循页面策略（编辑器/管理页需防误退）；音量为全局统一状态。
- 大按钮主操作（Primary CTA）
  - 样式：高对比、圆角、大面积；同一层级同一颜色体系。
  - 交互：按下态/禁用态清晰；禁用需给出原因（例如“正在生成作品…”）。
- 家长门禁弹层：简单算术题
  - 触发：进入设置中的高风险项（清理/重置/导出等）或离开应用（可选）。
  - 交互：错误提示非羞辱性；支持换题；支持关闭返回。
- 新手引导浮层：仅首次出现，可跳过
  - 交互：分步引导关键动作（选模板→涂色→完成→入海）；可随时跳过且不再打扰。

## 关键状态
- 空状态：无作品时引导“去涂色”
  - 覆盖页面：`/gallery`、`/ocean`（当无入海作品时显示引导入口）。
- 加载状态：线稿加载、作品生成、海洋动物进入场景
  - 体验：骨架屏/进度提示；避免长时间无反馈。
- 错误状态：IndexedDB 不可用/存储空间不足（给出降级与清理入口）
  - 降级：允许“仅临时体验”（不保存）或引导开启存储权限/更换浏览器（视平台）。
  - 清理：提供到 `/settings` 的清理入口；必要时给出“导出后清理”的文案（若导出未实现则不展示）。
