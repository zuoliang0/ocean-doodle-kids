## 沟通记录（摘要）
- 产品名称：海洋伙伴
- 形态：Pad 横屏 Web/H5
- 用户体系：无需登录、即开即用
- 核心玩法：选择海洋动物模板 → 分区涂色完成 → 将完成动物送入动态海洋世界，与系统小鱼一起游动
- 产出要求：规划功能与页面；输出包含 PRD、功能规划、技术方案、页面规划、功能清单；`pages` 数组可直接写入 `pages.json`，且每个页面包含 `name/route/description/uiPrompt`（用于后续生成 UI 图片）

## 关键假设（可后续确认）
- 目标设备：iPad/国产平板的横屏浏览器与 WebView（建议 1024×768 起）
- 存储方式：本地保存作品与海洋世界数据（IndexedDB/LocalStorage），不依赖账号
- 模板形态：SVG/矢量分区（每个区域可点击填色），支持少量笔刷涂抹作为增强（可选）
- 海洋世界：2D 动画（Canvas/WebGL），完成动物以“游动角色”形式加入场景
